# passport-js-app
Passport login app using Nodejs, express, mongoDB and ejs as Template engine

# Technologies
. Nodejs\
. Express\
. MongoDB / MongoJS\
. Ejs\
. Bcrypt hashing\
. Passport-local\
. Bootstrap\

# Install dependencies
. Clone the repository\
. Run npm install\

# Run app localhost:3000
. node app or npm start
